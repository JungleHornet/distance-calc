Welcome to JungleCalculator!

It currently has the following functions:

- General Calculator
- Distance Formula:
    - 2D Distance Formula
    - 3D Distance Formula
- Triangle Operation:
    - Pythagorean Theorem
    - Orthocenter
    - Circumcenter
    - Centroid
    - Incenter
- Miscellaneous Functions:
    - Radical Simplifier

To use, follow the console intructions.
